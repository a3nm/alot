Getting Account information¶

alot.account.AccountManager¶

class alot.account.AccountManager¶

Easy access to all known accounts

get_accounts()¶

return known accounts

Return type:list of account.Account
get_account_by_address(address)¶

returns account for given email address

Parameters:address (string) – address to look up
Return type:account.Account or None
get_main_addresses()¶

returns addresses of known accounts without its aliases

get_addresses()¶

returns addresses of known accounts including all their aliases

alot.account.Account¶

class alot.account.Account¶

Datastructure that represents an email account. It manages this account’s settings, can send and store mails to maildirs (drafts/send)

address¶

this accounts main email address

aliases¶

list of alternative addresses

gpg_key¶

gpg fingerprint. CURRENTLY IGNORED

realname¶

real name used to format from-headers

send_mail(mail)¶

sends given mail

Parameters:mail (email.message.Message or string) – the mail to send
Returns:None if successful and a string with the reason for failure otherwise
signature¶

path to a signature file to append to outgoing mails.

signature_filename¶

filename of signature file in attachment

store_draft_mail(mail)¶

stores mail as draft if draft_box is set

Parameters:mail (email.message.Message or string) – the mail to store
store_mail(mbx, mail)¶

stores given mail in mailbox. if mailbox is maildir, set the S-flag.

Parameters:
  • mbx (mailbox.Mailbox) – mailbox to use
  • mail (email.message.Message or string) – the mail to store
store_sent_mail(mail)¶

stores mail in send-store if sent_box is set

Parameters:mail (email.message.Message or string) – the mail to store
