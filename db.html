Database Abstraction¶

alot.db.DBManager¶

class alot.db.DBManager(path=None, ro=False)¶

keeps track of your index parameters, can create notmuch.Query objects from its Database on demand and implements a bunch of database specific functions.

count_messages(querystring)¶

returns number of messages that match querystring

flush()¶

tries to flush all queued write commands to the index.

Exception :DatabaseROError if db is opened in read-only mode
Exception :DatabaseLockedError if db is locked
get_all_tags()¶

returns all tags as list of strings

get_message(mid)¶

returns the message with given id as alot.message.Message object

get_thread(tid)¶

returns the thread with given id as alot.db.Thread object

query(querystring)¶

creates notmuch.Query objects on demand

Parameters:querystring – The query string to use for the lookup
Returns:notmuch.Query – the query object.
search_thread_ids(querystring)¶

returns the ids of all threads that match the querystring This copies! all integer thread ids into an new list.

tag(querystring, tags, remove_rest=False)¶

add tags to all matching messages. Raises DatabaseROError if in read only mode.

Parameters:
  • querystring (str) – notmuch search string
  • tags (list of str) – a list of tags to be added
  • remove_rest (boolean) – remove tags from matching messages before tagging
Exception :

NotmuchError
